# Missile Defense Game

## Overview
A single-player missile defense game inspired by Red Alert. Player deploys defensive assets (SAM sites, interceptors, radar) and defends against incoming waves of missiles, drones, and aircraft.

## Current State
MVP prototype - building core mechanics:
- Click to launch interceptor missiles
- Missiles move across screen
- Top-down 2D perspective

## Tech Stack
- Engine: Godot 4
- Language: GDScript
- Renderer: Forward+

## Development Approach
Starting minimal - prove core interception mechanics work before adding:
- Enemy waves
- Defensive placement strategy
- Resource management
- Progressive difficulty
